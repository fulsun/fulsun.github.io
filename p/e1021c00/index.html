Spring Boot жіЁи§Ј

еҸ‘иЎЁдәҺ2021-06-09|жӣҙж–°дәҺ2021-07-16|2_жЎҶжһ¶Springboot
|еӯ—ж•°жҖ»и®Ў:1.7k|йҳ…иҜ»ж—¶й•ҝ:5еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

Spring

@Component

  • @ComponentжіЁи§ЈиЎЁжҳҺвјҖдёӘзұ»дјҡдҪңдёәз»„д»¶зұ»пјҢе№¶е‘ҠзҹҘSpringиҰҒдёәиҝҷдёӘзұ»еҲӣе»әbean
  • @Component пјҲ@ControllerгҖҒ@ServiceгҖҒ@RepositoryпјүйҖҡеёёжҳҜйҖҡиҝҮзұ»и·Ҝеҫ„жү«жҸҸжқҘвҫғеҠЁдҫҰжөӢд»ҘеҸҠвҫғеҠЁиЈ…й…ҚеҲ°Springе®№еҷЁдёӯгҖӮ

@Bean

  • @BeanжіЁи§Је‘ҠиҜүSpringиҝҷдёӘвҪ…жі•е°Ҷдјҡиҝ”еӣһвјҖдёӘеҜ№иұЎпјҢиҝҷдёӘеҜ№иұЎиҰҒжіЁеҶҢдёәSpringеә”вҪӨдёҠдёӢвҪӮдёӯзҡ„beanгҖӮ
  • йҖҡеёёвҪ…жі•дҪ“дёӯеҢ…еҗ«дәҶжңҖз»Ҳдә§вҪЈbeanе®һдҫӢзҡ„йҖ»иҫ‘пјҢе№¶дё”е®һдҫӢеҗҚе°ұжҳҜвҪ…жі•еҗҚгҖӮ
  • @BeanеҲҷеёёе’Ң@ConfigurationжіЁи§Јжҗӯй…ҚдҪҝвҪӨгҖӮ

дҪҝз”ЁйҖ»иҫ‘

  1. еҰӮжһңжғіе°Ҷз¬¬дёүвҪ…зҡ„зұ»еҸҳжҲҗз»„д»¶пјҢдҪ вјңжІЎжңүжІЎжңүжәҗд»Јз ҒпјҢд№ҹе°ұжІЎеҠһжі•дҪҝвҪӨ@ComponentиҝӣвҫҸвҫғеҠЁй…ҚзҪ®пјҢиҝҷз§Қж—¶еҖҷдҪҝвҪӨ@Beanе°ұвҪҗиҫғеҗҲйҖӮдәҶгҖӮ
  2. дёҚиҝҮеҗҢж ·зҡ„д№ҹеҸҜд»ҘйҖҡиҝҮxmlвҪ…ејҸжқҘе®ҡд№үгҖӮ
  3. еҸҰеӨ–@BeanжіЁи§Јзҡ„вҪ…жі•иҝ”еӣһеҖјжҳҜеҜ№иұЎпјҢеҸҜд»ҘеңЁвҪ…жі•дёӯдёәеҜ№иұЎи®ҫзҪ®еұһжҖ§гҖӮ

@Configuration

  • @Configuration ж ҮжіЁзҡ„зұ»пјҢ @Bean ж ҮжіЁзҡ„вҪ…жі•гҖӮ
  • @Configuration ж ҮжіЁзҡ„зұ»зӯүеҗҢдәҺвјҖдёӘxmlвҪӮд»¶пјҢ @Bean ж ҮжіЁзҡ„вҪ…жі•зӯүеҗҢдәҺxmlвҪӮд»¶вҫҘзҡ„вјҖ дёӘж Үзӯҫ

FullжЁЎејҸе’ҢLiteжЁЎејҸ

  • FullжЁЎејҸе’ҢLiteжЁЎејҸеқҮжҳҜй’ҲеҜ№дәҺSpringй…ҚзҪ®зұ»вҪҪвҫ”зҡ„пјҢе’Ңxmlй…ҚзҪ®вҪӮд»¶вҪҶе…ігҖӮ
  • еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҡеҲӨж–ӯжҳҜFullжЁЎејҸ or LiteжЁЎејҸзҡ„еүҚжҸҗжҳҜпјҢвҫёе…ҲдҪ еҫ—жҳҜдёӘе®№еҷЁз»„д»¶гҖӮ
  • еҰӮжһңжҳҜеңЁе…¬еҸёзҡ„дёҡеҠЎеҠҹиғҪ/жңҚеҠЎдёҠеҒҡејҖеҸ‘пјҢдҪҝвҪӨFullжЁЎејҸ
  • еҰӮжһңдҪ жҳҜдёӘе®№еҷЁејҖеҸ‘иҖ…пјҢжҲ–иҖ…дҪ еңЁејҖеҸ‘дёӯй—ҙд»¶гҖҒйҖҡвҪӨз»„д»¶зӯүпјҢйӮЈд№ҲдҪҝвҪӨLiteжЁЎејҸжҳҜвјҖз§Қжӣҙиў«жҺЁиҚҗзҡ„вҪ…ејҸпјҢе®ғеҜ№ Cloud NativeжӣҙдёәеҸӢеҘҪ

LiteжЁЎејҸ

  • е®ҳж–№е®ҡд№үдёәпјҡеңЁжІЎжңүж ҮжіЁ@Configurationзҡ„зұ»йҮҢйқўжңү@Beanж–№жі•е°ұз§°дёәLiteжЁЎејҸзҡ„й…ҚзҪ®гҖӮ

  • йҖҸиҝҮжәҗз ҒеҶҚзңӢиҝҷдёӘе®ҡд№үжҳҜдёҚе®Ңе…ЁжӯЈзЎ®зҡ„пјҢиҖҢеә”иҜҘжҳҜжңүеҰӮдёӢcaseеқҮи®ӨдёәжҳҜLiteжЁЎејҸзҡ„й…ҚзҪ®зұ»пјҡ

    • зұ»дёҠж ҮжіЁжңү@ComponentжіЁи§Ј
    • зұ»дёҠж ҮжіЁжңү@ComponentScanжіЁи§Ј
    • зұ»дёҠж ҮжіЁжңү@ImportжіЁи§Ј
    • зұ»дёҠж ҮжіЁжңү@ImportResourceжіЁи§Ј
    • иӢҘзұ»дёҠжІЎжңүд»»дҪ•жіЁи§ЈпјҢдҪҶзұ»еҶ…еӯҳеңЁ@Beanж–№жі•
    • д»ҘдёҠcaseзҡ„еүҚжҸҗеқҮжҳҜзұ»дёҠжІЎжңүиў«ж ҮжіЁ@Configuration
    • еңЁSpring 5.2д№ӢеҗҺж–°еўһдәҶдёҖз§Қcaseд№ҹз®—дҪңLiteжЁЎејҸпјҡ
      • ж ҮжіЁжңү@Configuration(proxyBeanMethods = false)пјҢжіЁж„ҸпјҡжӯӨеҖјй»ҳи®ӨжҳҜtrueе“ҰпјҢйңҖиҰҒжҳҫзӨәж”№дёәfalseжүҚз®—жҳҜLiteжЁЎејҸ

  • з»Ҷеҝғзҡ„дҪ дјҡеҸ‘зҺ°пјҢиҮӘSpring5.2пјҲеҜ№еә”Spring Boot 2.2.0пјүејҖе§ӢпјҢеҶ…зҪ®зҡ„еҮ д№ҺжүҖжңүзҡ„@Configurationй…ҚзҪ®зұ»йғҪиў«дҝ®ж”№дёәдәҶ@Configuration(proxyBeanMethods = false)пјҢзӣ®зҡ„дҪ•дёәпјҹзӯ”пјҡд»ҘжӯӨжқҘйҷҚдҪҺеҗҜеҠЁж—¶й—ҙпјҢдёәCloud Nativeз»§з»ӯеҒҡеҮҶеӨҮгҖӮ

  • LiteжЁЎејҸдјҳзјәзӮ№

    • дјҳзӮ№пјҡ

      • иҝҗвҫҸж—¶дёҚеҶҚйңҖиҰҒз»ҷеҜ№еә”зұ»вҪЈжҲҗCGLIBвјҰзұ»пјҢжҸҗвҫјдәҶиҝҗвҫҸжҖ§иғҪпјҢйҷҚдҪҺдәҶеҗҜеҠЁж—¶й—ҙ,
      • иҜҘй…ҚзҪ®зұ»еҸҜд»ҘеҪ“дҪңвјҖдёӘжҷ®йҖҡзұ»дҪҝвҪӨпјҡд№ҹе°ұжҳҜиҜҙ@BeanвҪ…жі• еҸҜд»ҘжҳҜprivateгҖҒеҸҜд»ҘжҳҜfinal

    • зјәзӮ№пјҡ

      • дёҚиғҪеЈ°жҳҺ@Beanд№Ӣй—ҙзҡ„дҫқиө–пјҢд№ҹе°ұжҳҜиҜҙдёҚиғҪйҖҡиҝҮвҪ…жі•и°ғвҪӨжқҘдҫқиө–е…¶е®ғBean
      • е…¶е®һиҝҷдёӘзјәзӮ№иҝҳеҘҪпјҢеҫҲе®№жҳ“вҪӨе…¶е®ғвҪ…ејҸвҖңејҘиЎҘвҖқпјҢвҪҗеҰӮпјҡжҠҠдҫқиө–Beanж”ҫиҝӣвҪ…жі•вјҠеҸӮвҫҘеҚіеҸҜ

  • жҖ»з»“пјҡ

    • LiteжЁЎејҸдёӢпјҢй…ҚзҪ®зұ»жң¬вҫқдёҚдјҡиў«CGLIBеўһејәпјҢж”ҫиҝӣIoCе®№еҷЁеҶ…зҡ„е°ұжҳҜжң¬е°Ҡ
    • иҜҘжЁЎејҸдёӢпјҢеҜ№дәҺеҶ…йғЁзұ»жҳҜжІЎжңүйҷҗеҲ¶зҡ„пјҡеҸҜд»ҘжҳҜFullжЁЎејҸжҲ–иҖ…LiteжЁЎејҸ
    • иҜҘжЁЎејҸдёӢпјҢй…ҚзҪ®зұ»еҶ…йғЁдёҚиғҪйҖҡиҝҮвҪ…жі•и°ғвҪӨжқҘеӨ„зҗҶдҫқиө–пјҢеҗҰеҲҷжҜҸж¬ЎвҪЈжҲҗзҡ„йғҪжҳҜвјҖдёӘж–°е®һдҫӢвҪҪе№¶вҫ®IoCе®№еҷЁеҶ…зҡ„еҚ•дҫӢ
    • иҜҘжЁЎејҸдёӢпјҢй…ҚзҪ®зұ»е°ұжҳҜвјҖжҷ®йҖҡзұ»еҳӣпјҢжүҖд»Ҙ@BeanвҪ…жі•еҸҜд»ҘдҪҝвҪӨprivate/finalзӯүиҝӣвҫҸдҝ®йҘ°пјҲstaticвҫғз„¶д№ҹжҳҜйҳ”д»Ә зҡ„пјү

FullжЁЎејҸ

  • еңЁеёёв»…зҡ„еңәжҷҜдёӯпјҢ @Bean вҪ…жі•йғҪдјҡеңЁж ҮжіЁжңү @Configuration зҡ„зұ»дёӯеЈ°жҳҺпјҢд»ҘзЎ®дҝқжҖ»жҳҜдҪҝвҪӨвҖңFullжЁЎејҸвҖқпјҢиҝҷд№ҲвјҖжқҘпјҢдәӨеҸүвҪ…жі•еј•вҪӨдјҡиў«йҮҚе®ҡеҗ‘еҲ°е®№еҷЁзҡ„вҪЈе‘Ҫе‘Ёжңҹз®ЎзҗҶпјҢжүҖд»Ҙе°ұеҸҜд»ҘжӣҙвҪ…дҫҝзҡ„з®ЎзҗҶBeanдҫқиө–гҖӮ

  • дҪ•ж—¶дёәFullжЁЎејҸ

    1. ж ҮжіЁжңү@ConfigurationжіЁи§Јзҡ„зұ»иў«з§°дёәfullжЁЎејҸзҡ„й…ҚзҪ®зұ»гҖӮ
    2. вҫғSpring5.2еҗҺиҝҷеҸҘиҜқж”№дёәдёӢвҫҜиҝҷж ·жҲ‘и§үеҫ—жӣҙдёәзІҫзЎ®дәӣпјҡ
      • ж ҮжіЁжңү@ConfigurationжҲ–иҖ…@Configuration(proxyBeanMethods = true)зҡ„зұ»иў«з§°дёәFullжЁЎејҸзҡ„й…ҚзҪ®зұ»
      • еҪ“з„¶е–ҪпјҢproxyBeanMethodsеұһжҖ§зҡ„й»ҳи®ӨеҖјжҳҜtrueпјҢжүҖд»ҘвјҖиҲ¬йңҖиҰҒFullжЁЎејҸжҲ‘д»¬еҸӘйңҖиҰҒж Ү@ConfigurationжіЁи§ЈеҚіеҸҜ

  • FullжЁЎејҸдјҳзјәзӮ№

    • дјҳзӮ№пјҡ
      • еҸҜд»ҘвҪҖжҢҒйҖҡиҝҮеёёи§„Javaи°ғвҪӨзӣёеҗҢзұ»зҡ„@BeanвҪ…жі•вҪҪдҝқиҜҒжҳҜе®№еҷЁеҶ…зҡ„BeanпјҢиҝҷжңүж•Ҳи§„йҒҝдәҶеңЁвҖңLiteжЁЎејҸвҖқдёӢж“ҚдҪңж—¶йҡҫд»Ҙи·ҹиёӘзҡ„з»Ҷеҫ®й”ҷиҜҜгҖӮзү№еҲ«еҜ№дәҺиҗҢж–°зЁӢеәҸе‘ҳпјҢиҝҷдёӘзү№зӮ№еҫҲжңүж„Ҹд№ү
    • зјәзӮ№пјҡ
      • иҝҗвҫҸж—¶дјҡз»ҷиҜҘзұ»вҪЈжҲҗвјҖдёӘCGLIBвјҰзұ»ж”ҫиҝӣе®№еҷЁпјҢжңүвјҖе®ҡзҡ„жҖ§иғҪгҖҒж—¶й—ҙејҖй”ҖпјҲиҝҷдёӘејҖй”ҖеңЁSpring Bootиҝҷз§ҚжӢҘжңүвјӨйҮҸй…ҚзҪ®зұ»зҡ„жғ…еҶөдёӢжҳҜдёҚе®№еҝҪи§Ҷзҡ„пјҢиҝҷд№ҹжҳҜдёәдҪ•Spring 5.2ж–°еўһдәҶproxyBeanMethodsеұһжҖ§зҡ„жңҖзӣҙжҺҘеҺҹеӣ пјү
      • жӯЈеӣ дёәиў«д»ЈзҗҶдәҶпјҢжүҖд»Ҙ@BeanвҪ…жі• дёҚеҸҜд»ҘжҳҜprivateгҖҒдёҚеҸҜд»ҘжҳҜfinal

SpringBoot

@ConfigurationProperties

  • @ConfigurationPropertiesжҳҜspringbootжҸҗдҫӣиҜ»еҸ–й…ҚзҪ®вҪӮд»¶зҡ„вјҖдёӘжіЁи§ЈгҖӮ

  • е…¶еҜ№еә”зҡ„beanзҡ„еҗҺзҪ®еӨ„зҗҶеҷЁдёә ConfigurationPropertiesBindingPostProcessor

  • е®ғжҳҜе®һзҺ°дәҶBeanPostProcessorжҺҘвјқпјҢеңЁbeanиў«е®һдҫӢеҢ–еҗҺпјҢдјҡи°ғвҪӨеҗҺзҪ®еӨ„зҗҶпјҢйҖ’еҪ’зҡ„жҹҘжүҫеұһжҖ§пјҢйҖҡиҝҮеҸҚе°„жіЁвјҠеҖј

  • еҜ№вјӨеӨҡж•°еұһжҖ§вҪҪвҫ”ејәеҲ¶йңҖжҸҗдҫӣе…¶setterе’ҢgetterвҪ…жі•гҖӮ дҪҶжҳҜеұһжҖ§еҗҚз§°дёҚиҰҒжұӮвјҖе®ҡзӣёеҗҢпјҢеҸӘйңҖдҝқиҜҒй…ҚзҪ®вҪӮд»¶зҡ„еұһжҖ§е’ҢsetterвҪ…жі•еҗҚзӣёеҗҢеҚіеҸҜгҖӮ

  • дёҚиҝҮ@ConfigurationPropertiesжіЁи§Ј, еҸӘеҸҜд»ҘжҠҠеә”вҪӨдё»й…ҚзҪ®вҪӮд»¶зҡ„еҶ…е®№иҜ»еҸ–иҝӣжқҘ, иҰҒжғіз»ҷBeanиөӢеҖј, еҸӘиғҪиҜ»еҸ–дё»й…ҚзҪ®вҪӮд»¶,е°ұжҳҜapplication.propertiesжҲ–иҖ…application.ymlвҪӮд»¶

  • иӢҘжҲ‘д»¬жғівҫғвј°еҶҷвјҖдёӘй…ҚзҪ®вҪӮд»¶жқҘз»ҷbeanиөӢеҖј,жҲ‘д»¬е°ұеҸҜд»ҘдҪҝвҪӨиҜҘжіЁи§Ј@PropertyResource

  • жҲ‘д»¬еңЁresourceвҫҘиҫ№ж–°е»әвјҖдёӘstudent.propertiesвҪӮд»¶,е№¶жҠҠд№ӢеүҚдё»й…ҚзҪ®вҪӮд»¶вҫҘзҡ„еҶ…е®№еүӘиҙҙиҝӣжқҘ,е№¶еңЁbeanдёҠиҫ№еҠ  дёҠжіЁи§Ј

    жіЁвјҠbeanзҡ„дёүз§ҚвҪ…ејҸ

  • @ConfigurationPropertiesиҜ»еҸ–й…ҚзҪ®вҪӮд»¶, жіЁвјҠbeanзҡ„вҪ…ејҸ

    1. й…ҚеҗҲ@ComponentжіЁи§ЈзӣҙжҺҘиҝӣвҫҸжіЁвјҠпјҲдё»иҰҒпјү

    2. дҪҝвҪӨ@EnableConfigurationProperties,йҖҡеёёй…ҚзҪ®еңЁж Үжңү@Configurationзҡ„зұ»дёҠ,еҪ“з„¶е…¶д»–@ComponentжіЁ и§Јзҡ„жҙҫвҪЈзұ»д№ҹеҸҜд»ҘпјҢдёҚиҝҮдёҚжҺЁиҚҗ.

      • @EnableConfigurationPropertiesзҡ„дҪңвҪӨ: дҪҝ@ConfigurationPropertiesжіЁи§Јзҡ„зұ»вҪЈж•Ҳ,е№¶е°Ҷж ҮжіЁ@ConfigurationPropertiesзҡ„зұ»жіЁеҶҢеҲ°springе®№еҷЁдёӯ
      • @EnableConfigurationProperties({DocumentServerProperties.class}) еҸӘжҳҜи®©DocumentServerPropertiesзұ»з”ҹж•ҲпјҢе°Ҷе…¶жіЁе…ҘеҲ°springе®№еҷЁдёӯ

    3. дҪҝвҪӨ@BeanвҪ…ејҸеңЁж Үжңү@Configurationзҡ„зұ»иҝӣвҫҸжіЁвјҠ,иҝҷз§ҚвҪ…ејҸйҖҡеёёеҸҜд»ҘвҪӨеңЁеҜ№з¬¬дёүвҪ…зұ»иҝӣвҫҸй…ҚзҪ®еұһжҖ§жіЁеҶҢ
