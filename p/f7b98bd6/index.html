Hyper-V搭建centos8

发表于2018-09-28|更新于2021-07-16|3_LinuxCentos
|字数总计:460|阅读时长:1分钟|阅读量:

Hyper-V简介

Hyper-v 是 Microsoft 的硬件虚拟化产品。 它允许用户创建和运行计算机的软件版本（称为虚拟机）。 每个虚拟机的行为类似于一台运行操作系统和程序的完整计算机。
Hyper-v 在各自隔离的空间中运行每个虚拟机，这意味着用户可以同时在同一个硬件上运行多个虚拟机。

开启Hyper-V支持

  1. win10下使用win + s快捷键打开搜索页面，输入“控制面板”。
  2. 进入控制面板中的程序页面->程序和功能页面->启用或关闭windows功能。
  3. 进入启动或关闭windows功能页面后。开启 Hyper-V。

搭建centos8虚拟机环境

下载镜像

下载最新版的centos8镜像：http://mirror.layeronline.com/centos/8.0.1905/isos/x86_64/CentOS-8-x86_64-1905-dvd1.iso

新建虚拟机

进入Hyper-V管理器，新建虚拟机。

  1. 进入向导。
  2. 指定名称和位置。
  3. 指定代数。
    • 第一代适用Linux
    • 第二代适用Windows10等系统
  4. 分配内存：启用动态分配会占用宿主机的内存
  5. 网络设置–这边使用内置的默认网络。
  6. 选择虚拟磁盘–创建一个新的硬盘空间，专属于该虚拟机使用。
  7. 选择iso进行安装。点击完成后此时会在列表中出现刚才创建的虚拟机。连接并启动。

安装配置

  1. 选择第一个进行安装，等待加载完成。
    ![](index.zh-CN/20191216141939035_11224.png)
  2. 选择语言。
  3. 分区。
  4. 软件选择。
  5. 网络和主机设置
  6. 开始安装。进入安装页面，此时需要设置密码。
  7. 设置完成后进行等待安装完成。

  • 注意： 等安装完成后进行启动的时候要先进行弹出DVD驱动器，否则会重新安装。
