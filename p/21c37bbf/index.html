    2. 6位16进制数据表示颜色(二位相等时可以使3位)

    3. #000: 黑色 # fff:白色

文字样式

  1. font-size: 字体大小;

  2. 具体像素

  3. 百分比(对比父级)

  4. font-family:字体类型(不同类型,逗号隔开)

  5. 先英文

  6. 后中文

  7. font-weight:字体的权重

  8. normal   :　 默认值。正常的字体。相当于 400 。声明此值将取消之前任何设置

  9. bold     :　 粗体。相当于 700 。也相当于 b 对象的作用

  10. bolder   :　 比 normal 粗

  11. lighter  :　 比 normal 细

  12. 100-900

宽高样式

  1. width: 宽

  2. height:高

  3. px

  4. %

使用百分比需要确定父级的宽高

背景样式

  1. background-color:背景颜色;

  2. background-image:背景图片

  3. background-repeat:背景重复

repeat     :　 默认值。背景图像在纵向和横向上平铺

no-repeat  :　 背景图像不平铺

repeat-x   :　 背景图像仅在横向上平铺

repeat-y   :　 背景图像仅在纵向上平铺

  1. background-position:背景定位

列表样式

  1. list-style-type: 列表前的图形

  2. none;

浮动样式

给一个标签添加浮动后,会脱离标准文档流,

left浮动:挨着父级容器的左边框或者已经浮动的容器的右边框停止浮动

子元素的浮动之和不能超过父级的大小

盒子模型

夏敏属性都分为上下左右四部分

可以简写:

4个:顺时针 上 右 下 左

2个: 上下   左右

1个: 上下左右
